How to get working savings today

Start with one code and a simple cart. Apply the code before you add extras. If it fails, fix the cart, then try again. Keep it clean and quick.

  1. Open the official hub, apply one code only.
  2. Remove excluded items like gift cards or alcohol when the terms say so.
  3. Meet the minimum basket size if required by the offer.
  4. Try pickup if delivery fees are excluded by the code.
  5. Sign out, then sign in again if the cart acts strange.

Fix checkout errors quickly

  • Remove excluded items, then reapply the code
  • Clear the cart, sign out, then sign in again
  • Switch to pickup if delivery is excluded by the offer
  • Use only one code, stacking usually blocks checkout

Important reminders

  • Values and availability change by retailer and location
  • Some stores require a linked loyalty account
  • Codes may expire or be account limited
  • Always check the final price before you confirm
